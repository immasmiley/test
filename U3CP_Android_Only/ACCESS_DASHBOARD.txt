🔧 TROUBLESHOOTING:

If dashboard doesn't load:
1. Make sure you're using: file:///sdcard/native_dashboard.html
2. Try a different browser (Chrome, Firefox, Samsung Internet)
3. Check if the file exists: file:///sdcard/ (should see native_dashboard.html)

If Termux doesn't work:
1. Open Termux app manually
2. Run: pkg update -y
3. Run: pkg install python -y
4. Run: cd /sdcard && ./start_u3cp_termux.sh

If Python not found in Termux:
1. In Termux, run: pkg install python -y
2. Wait for installation to complete
3. Run: python --version (to verify)
4. Then run the U3CP system
